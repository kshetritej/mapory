# Mapory – Interactive Travel Memories Map

Mapory is a platform where users can pin their travel memories on a map. Users can upload image, write short stories, and mark the exact location of their experiences. Built with **Next.js**, **Leaflet**, and **Supabase**.

## Features

- Drop pins on a map to record memories  
- Upload image and add short messages  
- View other users’ Mapories interactively  
- User authentication via Supabase  

## Tech Stack

- **Frontend:** Next.js + React + Tailwind CSS  
- **Backend:** Supabase (Auth, Storage, Database)  
- **Map:** Leaflet (OpenStreetMap)
